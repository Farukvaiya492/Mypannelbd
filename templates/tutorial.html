Step by Step Guide

1

Visit Telegram Web

Open this link in your browser:

https://my.telegram.org
2

Log In

Log in with your Telegram phone number. You'll receive a code in your Telegram app - enter that code.

Important: If you have 2FA (Two-Factor Authentication) enabled on your Telegram account, you'll need to enter your password.

3

Go to API Development Tools

After logging in, click on the "API development tools" option.

4

Create New Application

Fill in the following information to create a new application:

  • App title: Any name (e.g., My Bot)
  • Short name: Short name (e.g., mybot)
  • Platform: Select something (e.g., Desktop)
  • Description: (Optional - you can leave it blank)

Then click the "Create application" button.

5

Copy API ID and API Hash

Once the application is created, you will see:

API ID

A number (e.g., 12345678)

API Hash

A long hex code (e.g., 0123456789abcdef0123456789abcdef)

Security Warning: Never share this API ID and Hash with anyone! These are the keys to your Telegram account.

Go to Dashboard

Go to "Telegram Bot Manager" Section

On the right side of the main page, you will see a form called "Telegram Bot Manager". There:

API ID Field

Paste your copied API ID number

Example: 12345678

API Hash Field

Paste your copied API Hash code

Example: 0123456789abcdef0123456789abcdef

Phone Number Field

Enter your Telegram phone number (start with +)

Example: +8801712345678

Start Bot Button

Click the "Start Bot" button

You will receive a Verification Code in your Telegram. If 2FA is enabled, you'll need to enter your password.

Success! Your Telegram Bot is now running!

Your bot will now send and receive messages on your Telegram account.
